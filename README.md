# MJCompiler
University project in program compilers - compiler for Micro Java

Run configurations:
- LexerGenerator: generates lexer code in Yylex.java from mjlexer.flex file
- MJTest: tests lexer on example Micro Java program
- ParserGenerator: generates parser code in MJParser.java from mjparser.cup file
- Compile MJ Source: compiles example Micro Java program and outpus obj file
- Disasm: outputs disassembled obj file
- Run: executes obj file generated from example Micro Java program
